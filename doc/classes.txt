
Перечень классов и их назначение

В данном файле перечисляются классы и краткое описание об их назначении

=== Библиотечные классы ===

AVCEncoder - класс-оболочка вокруг AVCodecContext для кодирования медиа
+-- AVCVideoEncoder - класс для кодирования видео-потоков
+-- AVCAudioEncoder - класс для кодирования аудио-потоков

AVCDecoder - класс-оболочка вокруг AVCodecContext для декодирования медия
+-- AVCVideoDecoder - класс для декодирвоания видео-потоков
+-- AVCAudioDecoder - класс для декодирования аудио-потоков

=== Классы сервера avc_server ===

AVCEngine - главный класс сервера
AVCScene - сцена видеоконференции
AVCChannel - класс представляющий клиента работающего по протоколу MultiStream
AVCHttp - класс представляющий клиента работающего по протоколу HTTP
